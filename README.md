## 一、工作流架构总览

我设计的 7 步标准化流水线，每一步输入、工具、产出均为结构化数据，可审计、可复用、可规模化：

```
[①创意] → [②剧本] → [③角色] → [④分镜] → [⑤视觉] → [⑥音频] → [⑦合成]
 一句话       JSON       角色设定     画面Prompt    生图+视频     TTS+BGM      竖屏mp4
```

| 步骤 | 环节 | 输入 | 推荐工具 | 产出 |
|------|------|------|------|------|
| ① | 创意输入 | 一句话主题 / 热点词 | 人工 + DeepSeek | 3-5 个剧情梗概 |
| ② | 剧本生成 | 选定梗概 | **DeepSeek API** | JSON 剧本（场景/对白/情绪标记） |
| ③ | 角色设计 | 剧本角色列表 | DeepSeek + 豆包/即梦 | 角色设定文档 + 定装参考图 |
| ④ | 分镜设计 | 剧本 + 角色设定 | **DeepSeek API** | 逐镜画面描述 + 运镜建议 + 生图 Prompt |
| ⑤ | 视觉生成 | 分镜 Prompt | 豆包/即梦(免费额度) + 可灵 | 关键帧图片 → 5s 视频片段 |
| ⑥ | 音频合成 | 剧本对白 + 情绪标记 | **Edge-TTS**(免费) / 豆包 TTS | 逐句配音 + BGM 铺底 |
| ⑦ | 合成导出 | 视频片段 + 配音 + BGM | 剪映 / FFmpeg | 1080×1920 竖屏 mp4 |

### 为什么是 7 步而不是端到端

端到端模型（如 Sora 直接文生视频）的问题是：输出黑箱，出问题无法定位。7 步流水线每一步产出可审计、可反查、可精准修正。**可控性是工业化的基础。**

## 二、技术选型逻辑

选型原则：在保证质量的前提下，将单品试错成本压到最低。

| 环节 | 首选方案 | 备选方案 | 选型理由 |
|------|------|------|----------|
| 剧本/分镜 | **DeepSeek** | GPT-4o / Claude | 中文网文语境最强，JSON 输出稳定，成本极低 |
| 关键帧生图 | **豆包 / 即梦** | Midjourney / SD | 免费额度，中文 Prompt 友好，字节生态 |
| 图生视频 | **可灵 (Kling)** | Seedance 2.0 | 国产视频生成顶尖，有免费额度 |
| 配音 TTS | **Edge-TTS** | GPT-SoVITS | 完全免费，微软引擎 |
| 字幕生成 | **剪映自动字幕** | Faster-Whisper | 免费，准确率高 |
| 剪辑合成 | **剪映 / FFmpeg** | — | 免费工具覆盖全需求 |
| 工作流编排 | **Coze.cn** | — | 可视化拖拽，免费，多 Agent |

### DeepSeek vs GPT-4o 对比

| 维度 | DeepSeek | GPT-4o |

## 三、成本控制策略

核心理念：**一次生成，永久复用。** 首条跑通全流程后，60% 素材可跨集复用，边际成本锐减。

### 成本结构（均为预估，实际需根据项目具体用量核算）

| 环节 | 免费额度来源 | 免费方式 | 常规付费参考 | 量产优化方向 |
|------|-------------|---------|-------------|----------|
| 剧本 | DeepSeek 注册赠送 | 按 token 计费 | ¥0.02-0.05/条 | 精确 Prompt，一次出稿 |
| 角色生图 | 豆包/即梦每日免费 | 每日数十张 | ¥0.05/张 | 角色定装只生成一次 |
| 关键帧 | 豆包/即梦每日免费 | 每日数十张 | ¥0.05/张 | 同场景复用背景 |
| 图生视频 | 可灵/豆包免费额度 | 每日数次 | ¥0.5-2/条 | 分段生成+重排 |
| 配音 | Edge-TTS → **无限免费** | — | 0 | — |
| BGM | 免版权曲库 → **永久免费** | — | 0 | — |
| 字幕 | 剪映 → **免费** | — | 0 | — |
| 剪辑 | 剪映/FFmpeg → **免费** | — | 0 | — |

**全链路 7 个环节中 6 个可完全免费覆盖。**

### 降本路径（预估模型）

```
首条（验证流程）：核心开销在试错，预计 ¥3-5
第 5 条（库建立）：角色/场景资产可复用，预计 ¥1-2
第 10 条+（规模化）：素材复用率 >60%，预计 ¥0.5-1
第 20 条+（工业化）：模板化批量生产，边际成本趋近 ¥0.3
```

> 注：以上为基于工具定价的理论估算，未计入人力成本。实际成本需根据项目规格和产出频率实测后校准。
