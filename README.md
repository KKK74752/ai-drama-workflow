## 一、工作流架构总览

从创意到成片的 **7 步 AI 短剧标准化生产流水线**，每一步的输入、工具、产出均为结构化数据，可审计、可复用、可规模化。

```
[①创意] → [②剧本] → [③角色] → [④分镜] → [⑤视觉] → [⑥音频] → [⑦合成]
 一句话       JSON       角色设定     画面Prompt    生图+视频     TTS+BGM      竖屏mp4
```

| 步骤 | 环节 | 输入 | 工具 | 产出 |
|------|------|------|------|------|
| ① | 创意输入 | 一句话主题 / 热点词 | 人工 + DeepSeek 扩展 | 3-5 个剧情梗概 |
| ② | 剧本生成 | 选定梗概 | **DeepSeek API** | JSON 剧本（场景/对白/情绪标记） |
| ③ | 角色设计 | 剧本角色列表 | DeepSeek + 豆包/即梦生图 | 角色设定文档 + 定装参考图 |
| ④ | 分镜设计 | 剧本 + 角色设定 | **DeepSeek API** | 逐镜画面描述 + 运镜建议 + 生图 Prompt |
| ⑤ | 视觉生成 | 分镜 Prompt | 豆包/即梦(免费) + 可灵(付费) | 关键帧图片 → 5s 视频片段 |
| ⑥ | 音频合成 | 剧本对白 + 情绪标记 | **Edge-TTS**(免费) / 豆包 TTS | 逐句配音 + BGM 铺底 |
| ⑦ | 合成导出 | 视频片段 + 配音 + BGM | 剪映(手动精修) / FFmpeg(批量) | 1080×1920 竖屏 mp4 |

### 为什么是 7 步而不是端到端

端到端模型（如 Sora 直接文生视频）不透明，出问题不知道改哪里。七步流水线每一步产出可审计、可反查、可精准修正。**可控性才是工业化的基础。**

## 二、技术选型与工具链

**选型标准：在保证质量的前提下，把单条短剧试错成本压到最低。**

| 环节 | 首选 | 备选 | 选型理由 |
|------|------|------|----------|
| 剧本/分镜 | **DeepSeek** | GPT-4o / Claude | 中文网文语境最强，JSON 稳定，成本极低 |
| 关键帧生图 | **豆包 / 即梦** | Midjourney / SD | 免费额度，中文 Prompt 友好，字节生态 |
| 图生视频 | **可灵 (Kling)** | Seedance 2.0 | 国产视频生成顶尖，有免费额度 |
| 配音 TTS | **Edge-TTS** | GPT-SoVITS | 完全免费，微软引擎，音色自然 |
| 字幕 | **剪映自动字幕** | Faster-Whisper | 免费，准确率高 |
| 剪辑合成 | **剪映 / FFmpeg** | — | 免费 |
| 工作流编排 | **Coze.cn** | — | 可视化拖拽，免费，多 Agent |

### DeepSeek vs GPT-4o

| 维度 | DeepSeek | GPT-4o |
|------|----------|--------|
| 中文短剧理解 | ★★★★★ 网文语感极强 | ★★★☆☆ 翻译腔 |
| 单条成本 | ¥0.02-0.05 | ¥0.5-2.0 |
| JSON 结构化 | ★★★★☆ | ★★★★★ |
| API 兼容 | OpenAI 格式 | 原生 |

## 三、成本控制方案

**核心理念：一次生成，永久复用。**

第一条短剧跑通全流程可能需 1 小时，做到第 10 条时 **60% 素材可复用**，单条压缩到 15 分钟，成本降到 1 元以内。

| 环节 | 免费额度来源 | 免费额度 | 超出单价 | 量产优化 |
|------|-------------|---------|---------|----------|
| 剧本 | DeepSeek 注册赠送 | 充足 | ¥0.02/条 | 精确 Prompt 减少重试 |
| 角色生图 | 豆包/即梦每日免费 | 每日数十张 | ¥0.05/张 | 角色一次生成，全剧复用 |
| 关键帧 | 豆包/即梦每日免费 | 每日数十张 | ¥0.05/张 | 同场景复用背景 |
| 图生视频 | 可灵/豆包免费额度 | 每日数次 | ¥0.5-2/条 | 分段+重排 |
| 配音 | Edge-TTS | **无限免费** | — | — |
| BGM | 免版权曲库 | **永久免费** | — | — |
| 字幕 | 剪映 | **免费** | — | — |
| 剪辑 | 剪映/FFmpeg | **免费** | — | — |

**全链路 7 个环节中 6 个完全免费。**

### 成本递减

```
第 1 条：¥3-5（含试错）
第 5 条：¥1-2（角色/场景库建立）
第 10 条+：¥0.5-1（素材复用率 60%）
第 20 条+：边际成本 → ¥0.3
```

## 四、行业认知与自我定位

### 我不是"会用 AI 的剪辑师"

我是 **"能把 AI 短剧从零到一搭建起来的制片人"**。

| 剪辑师 | 制片人 |
|--------|--------|
| 执行单个环节 | 定义全链路 |
| 会用工具 | 会选工具、搭流水线、控成本 |
| 关注画面 | 关注成本×质量×效率三角平衡 |
| 单人干活 | 能带团队按标准执行 |

### AI 短剧不是炫技，是技术平权

过去做短剧需要编剧+分镜师+摄影师+剪辑师+配音师至少 5 人。现在一个人+一套 AI 工具链 = 零成本试错→快速验证→规模化量产。

从我 30+ 真实付费 PPT 订单验证的一个道理：**AI 不淘汰人，淘汰不会用 AI 的人。**

### 我能为团队带来什么

1. **技术落地** — 把"AI 做短剧"从概念变成标准化流程
2. **成本控制** — 免费额度策略，量产后边际成本极低
3. **质量保障** — 分层审计，每一步可检查可修正
4. **团队赋能** — 搭好工作流后非技术人员也能操作
